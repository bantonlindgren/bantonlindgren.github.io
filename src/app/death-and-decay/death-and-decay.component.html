Death and Decay

â€‹This is a scene I set up in Unreal Engine 4. My objective was to create a nice ambiance with shaders, particles and lighting to create an effect where everything dies as a the character gets close to it. In this project I wanted to play around with different ways in which to change the environment based on where the character was located. All 3D assets and textures are free resources (see credits at the bottom of the page).

Wall/ Floor shader

One of the most prominent parts of the scene is the wall/ floor material. I used two noise textures that I panned in different directions which I then stepped in order to get the masks for the rifts.

At first I used regular 2D textures but they didn't blend well between the walls and the floor. So I switched it up to use 3D noise textures and sampled them at the world position instead of the UV coordinates to get a seamless transition between the models.

In order to only spawn the rifts around the players I set up a parameter collection that saves the player world position through a blueprint. I read this position in the shader and use it create a spheremask with a falloff around it that I multiply with the noise mask.

Finally I set the textures and tiling up to be parameterized so I could just create separate material instances for the walls and floor while using the same material.

Candle flame

This is a particle system that utilized a candle flame texture. The material makes the flame sway back and forth a bit.

The system consists of two emitters, one for the flame and one for the smoke. In the candle blueprint I set up a sphere collider that reacts to the player pawn and sets the spawn rate of the emitters to enable the smoke and disable the flames when the character comes close (and vice versa when he leaves the trigger volume).

Molding fruit

Similar to the walls and floor I set up a spheremask with a falloff from the character position to the fruits to make it appear more and more moldy as the character gets closer to it.

I set up two parameterized colors for the moldy state. One base color and one for the spots. I lerp between them with a cloud texture and further I use the spheremask to lerp between the normal color and the mold color.

I push the vertices in towards the origin in the horizontal plane. In the Z axis I push the towards the bottom of the model with a vertical gradient as bias to push the top vertixes more than the bottom ones. I use the object bounds and position to calculate this gradient.

I also set up a particle system with flies that appear when the character gets close enough.

Withering flower

Similar to the fruit I made a shader for withering flowers.

The colors are faded out when the character gets closer to them and there is a bit of vertex displacement based on a cloud texture to give it a more wrinkled appearance.
